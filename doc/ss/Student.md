## 一覧取得

| メソッド | エンドポイント |
| -------- | -------------- |
| GET      | -              |

リクエスト

| 論理名 | 物理名 | 型  | 必須 | 桁  | 補足 |
| ------ | ------ | --- | ---- | --- | ---- |
| -      |        |     |      |     |      |

レスポンス

| 論理名     | 物理名      | 型           | 必須 | 桁  | 補足 |
| ---------- | ----------- | ------------ | ---- | --- | ---- |
| 生徒リスト | studentList | List(Object) |      |     |      |
| - 生徒番号 | studentId   | integer      | ◯    |     |      |
| - 苗字     | familyName  | string       | ◯    |     |      |
| - 名前     | firstName   | string       | ◯    |     |      |

レスポンスサンプル

```json
{
  "studentLists":[
    {
        "studentId": 1,
        "familyName": "佐藤",
        "firstName": "太郎"
    }
  ]
}
```

処理フロー

```mermaid
flowchart TD
    A1(("開始")) --> A2[" studentテーブルから全件取得 "]
    A2 --> Z(("終了"))
```
